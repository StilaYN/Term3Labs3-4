# Term3Labs3-4
## Общие задание
Вы должны работать в команде. Для этого начните с проектирования будущего приложения, разделите классы между членами бригады, договоритесь об интерфейсах, через которые ваши классы будут взаимодействовать. Исходный код должен находиться в репозитории Git, быть покрыт модульными тестами и документирован. Объекты должны поддерживать сериализацию и десериализацию. Должна быть возможность закрыть программу с сохранением состояния и затем запустить её из сохранённого состояния. ГПИ в этой ЛР не нужен.
## Лабораторная работа 3
Просмотрщик фотографий с возможностью их обработки. Должна быть возможность обрезать, поворачивать фотографии, составлять из них композиции (коллажи), добавлять надписи, менять отдельные пиксели. Реализуйте несколько фильтров, например, осветления, изменения цветов.
## Лабораторная работа 4
Добавить в разработанную в ходе лабораторной работы 3 программу графический пользовательский интерфейс на Windows Presentation Foundation или Avalonia UI. Для динамичных игр допустимо использование Unity.
